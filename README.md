Project Description
- A simple android quiz application designed in MVVM scalalbel folder architecture.
Features
-Start the quiz
-Answer to questions with the provided options
-Look at our results(Streak score,questins skipped,questions answered correctly,Retake).

Teach Stack
- Androi,Kotlin,XML,JSON

Folder Structure
-model
  -data
    -Quiz.Kt
  -network
    -ApiService.kt
    -RetrofitBuilder.kt
-Repository
  -QuizRepository.kt

-View
  -MainScreenActivity.kt
  -QuizFragment.kt
  -ResultFragment.kt

Api Usauage
-Included with get Api to get question list.
